# Creative Agency Portfolio Dashboard

An internal creative agency dashboard built with React and Vite for managing and showcasing agency projects, campaigns, and creative work.

The application allows users to:
- Add projects
- Search projects
- Organize agency work
- Display project cards in a clean dashboard layout

## Features

- Modern dashboard UI
- Responsive project grid
- Add new projects
- Search functionality
- Reusable React components
- Soft red and white agency branding
- Built with React + Vite

Installation & Setup
1. Clone the repository
git clone https://github.com/ckwangechi/creative-agency-portfolio.git
2. Navigate into the project folder
cd creative-agency-portfolio
3. Install dependencies
npm install
4. Start the development server
npm run dev
5. Open in browser

Vite will provide a local development URL such as:

http://localhost:5173
